# Correct Claims and Incorrect Claims

## Correct Claims

Use these claims when describing the project:

- This is a simulated MCP governance lab.
- This is a production-shaped public demo.
- The lab demonstrates AI tool governance patterns.
- The lab shows identity-aware policy evaluation.
- The lab shows risk-tiered MCP-style tool access.
- The lab shows a tool-call firewall before execution.
- The lab shows human approval before sensitive tool execution.
- The lab shows evidence replay and executive visibility.
- The lab includes a mock MCP execution adapter.
- The lab is suitable for client and portfolio demonstrations.

## Incorrect Claims

Do not claim:

- This is a production enforcement system.
- This is connected to real MCP servers.
- This processes real customer records.
- This processes real regulated data.
- This performs real enterprise authorization.
- This is SOC 2 certified.
- This is an enterprise IAM system.
- This guarantees compliance.
- This connects to real enterprise databases.
- This deploys production security controls.

## Safe Summary

SecureTheCloud MCP Governance Lab is a public, simulated, production-shaped demo showing how AI agents can be governed before MCP-style tool access is allowed.
