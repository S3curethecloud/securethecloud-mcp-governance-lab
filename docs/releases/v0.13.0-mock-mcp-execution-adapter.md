# Release v0.13.0 — Mock MCP Execution Adapter

## Summary

This release adds a governed mock MCP execution adapter to the SecureTheCloud MCP Governance Lab.

The lab now demonstrates the full AI tool governance flow from request intake through policy, firewall, approval, mock execution, evidence replay, and executive visibility.

## Added

- Mock MCP execution endpoint
- Governed execution result model
- Mock execution result builder
- Frontend execution adapter panel
- Execution status outcomes:
  - executed
  - blocked
  - requires_approval
  - redacted
  - escalated
- Evidence capture for mock execution attempts
- Dashboard updates after mock execution
- Public demo reset support from Phase 12

## Demo URL

https://securethecloud-mcp-governance-lab.fly.dev

## Correct Claim

This is a simulated MCP governance lab. It demonstrates the control pattern required before AI agents can safely access MCP-style tools.

## Boundary

No real MCP servers, customer records, regulated data, or enterprise systems are connected.
