## Short Description

Simulated MCP governance control plane for AI tool access, featuring policy decisions, tool-call firewalling, human approval, evidence replay, executive visibility, and governed mock execution.

## Portfolio Summary

SecureTheCloud MCP Governance Lab is a full-stack public demo showing how AI agents can be governed before they invoke MCP-style tools. It demonstrates identity context, data classification, risk tiering, policy decisions, firewall checks, approval routing, evidence replay, and mock execution outcomes.

## Technical Highlights

- FastAPI backend
- Next.js frontend
- Docker Compose local environment
- Fly.io public deployment
- Protected demo reset
- Mock MCP execution adapter
- Evidence replay timeline
- Executive governance dashboard
- Client-safe public demo boundary

## Public Demo

https://securethecloud-mcp-governance-lab.fly.dev

## Safe Claim

This is a simulated, production-shaped governance lab. It does not connect to real MCP servers, real customer records, regulated data, or enterprise systems.
